The system is dark, cinematic, and image-led. Use violet, pink, and cyan as luminous creative accents around prompts, generation states, and gallery cards. Keep panels readable and avoid low-contrast neon overload.

## Typography

Use a clean Chinese-first sans stack with high-impact display headings and compact product labels. Hero copy can be poetic and short. Product explanations should be direct and feature-led.

Recommended scale:

- Hero: 64-96px desktop, 44-58px mobile, 0.94-1.04 line-height.
- Section title: 36-56px, 1.02-1.12 line-height.
- Feature number: 40-64px, light weight or outline style.
- Card title: 18-26px, 1.18-1.3 line-height.
- Body: 15-18px, 1.55-1.75 line-height.
- Prompt labels: 12-14px.

## Layout

Use immersive full-viewport or near-full-viewport hero sections with a central prompt or CTA. Subsequent sections alternate between media showcase, numbered feature explanations, image grids, and intelligent canvas demos.

For creation tools, use a large canvas with floating controls, multi-layer thumbnails, prompt input, generation mode tabs, and precise operation chips such as redraw, expand, erase, cutout, and style transfer.

## Components

- Cinematic hero: dark media field, short title, primary CTA, prompt sample.
- Generation prompt bar: large rounded input, mode selector, generate button, optional image upload chip.
- Video feature section: numbered text block with media preview and CTA.
- Image gallery card: large visual slot, prompt caption, creator/action meta.
- Smart canvas: layered image board, floating tool palette, selected region outline.
- Inspiration community card: creator name, remix count, same-style action.
- Future plan banner: immersive full-width CTA with gradient border.

## Interaction

Motion and generation states matter. Use soft glow focus, progress chips, shimmer only inside media placeholders, hover zoom on inspiration cards, and selected-layer outlines on smart canvas tools. Keep CTA language action-oriented and Chinese-first when generating Chinese product UI.

## Boundaries

This package describes visual structure and AI-imaging behavior only. Do not copy Jimeng logos, generated images, creator names, screenshots, or exact marketing copy. Generate original creative UI while preserving the dark cinematic tone, Chinese prompt-first workflow, intelligent canvas controls, and inspiration-gallery rhythm of the reference page.
