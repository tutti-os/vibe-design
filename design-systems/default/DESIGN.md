# Vibe Default

> Category: Application

A quiet application design system for clear workbench interfaces, fast scanning, and high-confidence generated artifacts.

## Color

- Canvas: #f7f8fb
- Ink: #111827
- Panel: #ffffff
- Border: #d8dde8
- Accent: #2563eb

## Typography

Use Lexend Variable with Lexend and system UI fallbacks. Keep headings direct, medium-weight, and compact. Body copy should prioritize legibility over decoration.

## Layout

Use dense but breathable spacing, 8px radii, stable grid tracks, and visible hierarchy between navigation, workspace, and detail panels.

## Components

Buttons, cards, fields, and badges should use the token contract in `tokens.css`. Avoid raw colors outside the token block.
