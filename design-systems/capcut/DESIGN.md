# CapCut Creator Reference

> Category: Creator tool

A fast AI creator-suite system inspired by CapCut public editor and landing surfaces. Use it for video editing, social content creation, AI media generation, template galleries, and export workflows that need to feel energetic, direct, and easy for non-experts.

The visual rhythm should feel creator-focused and high momentum. Use clean light panels for onboarding and tool discovery, then switch to dark workspace panels for editing, timelines, preview monitors, and media bins.

## Typography

Use a modern sans stack with compact controls and clear product headings. Keep labels short, practical, and action-oriented. Avoid editorial flourish; this system is optimized for speed and scanning.

Recommended scale:

- Hero: 48-72px desktop, 36-48px mobile, 0.98-1.08 line-height.
- Tool section title: 28-44px, 1.08-1.18 line-height.
- Card title: 18-24px, 1.16-1.3 line-height.
- Body: 14-17px, 1.45-1.6 line-height.
- Control labels: 12-14px.
- Timeline timecode: 11-13px monospace.

## Layout

Landing and discovery pages use bold hero blocks, tabbed tool categories, feature cards, and template carousels. Editor surfaces use a four-zone composition: left asset/tool rail, central preview canvas, right AI or property panel, and bottom timeline.

Use stable dimensions for media cards, thumbnails, tracks, and export controls. Content creators should always see the next action: upload, generate, edit, caption, template, export, or share.

## Components

- Creator hero: large heading, compact promise, primary online CTA, secondary download CTA.
- Tool switcher: segmented tabs for image, video, templates, audio, and AI tools.
- Template card: thumbnail-first card with platform ratio badge and use count.
- AI prompt bar: prominent input with generate button and optional chips.
- Editor shell: dark preview stage, asset rail, property panel, bottom timeline.
- Timeline track: clip blocks, playhead, timecode, caption lane, audio lane.
- Export modal: format, aspect ratio, quality, platform presets, primary export action.
- Smart action chip: background remover, auto captions, text-to-speech, enhancer.

## Interaction

Interactions should be quick and obvious. Hover states lift cards slightly or brighten borders. Active tools use teal highlights. Timeline controls need precise selected, dragging, and trimming states. AI actions should show progress, queued states, and generated result cards.

## Boundaries

This package describes visual structure and creator-tool behavior only. Do not copy CapCut logos, screenshots, media assets, templates, or proprietary page copy. Generate original creator UI while preserving the approachable AI editing workflow, template density, and fast export orientation of the references.
