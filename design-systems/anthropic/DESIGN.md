# Anthropic Web Reference

> Category: Research product

A warm editorial research-and-product system inspired by Anthropic public pages and Claude Design guidance. Use it for AI research, safety, product education, collaborative design workspaces, and thoughtful enterprise pages that need calm authority.

The system is warm, not beige-only. Use parchment and off-white as the base, ink for hierarchy, muted clay for action emphasis, and dark panels for code or focused workspace regions. Avoid glossy gradients and saturated tech blues.

## Typography

Use a humanist or editorial serif for large brand moments when available, paired with a neutral sans for interface controls. If a serif is unavailable, use a high-quality system sans with generous tracking set to normal. Keep prose readable and quiet.

Recommended scale:

- Hero: 60-92px desktop, 42-56px mobile, 0.95-1.05 line-height.
- Research statement: 40-64px, 1.02-1.12 line-height.
- Section title: 28-44px, 1.08-1.18 line-height.
- Card title: 20-28px, 1.16-1.28 line-height.
- Body: 16-19px, 1.5-1.7 line-height.
- Meta: 12-14px, uppercase only when sparse.
- Workspace UI: 13-15px with compact, legible rhythm.

## Layout

Use a publication-like page structure: minimal navigation, large statements, measured content width, and generous vertical sections. News and research surfaces should use horizontal dividers, metadata columns, and large story cards rather than decorative tiles.

Claude Design-inspired workspaces should use a two-pane layout: conversation and controls on the left, canvas or artifact preview on the right. Inline comments, review states, and export actions should be visible but not dominant.

## Components

- Header: text mark, sparse navigation, grouped menus, one primary action.
- Research hero: large statement, one concise paragraph, restrained CTA row.
- Release list: title, summary, date, category, and understated read link.
- Story card: large media area or warm panel, strong title, muted metadata.
- Workspace shell: left chat rail, right canvas, top export/share controls.
- Comment marker: small numbered or dot marker with warm accent.
- Design-system panel: token swatches, typography samples, component references.
- Footer: broad multi-column taxonomy with small links and high information density.

## Interaction

Interactions should feel thoughtful and precise. Use underlines, subtle border changes, and warm highlight fills. Chat and canvas affordances should support iteration: selected element outline, inline comment badge, compact export menu, and version status.

## Boundaries

This package describes visual structure and design behavior only. Do not copy Anthropic or Claude logos, screenshots, illustrations, proprietary page copy, or exact layouts. Generate original content while preserving the warmth, research tone, restrained information architecture, and collaborative workspace patterns from the reference pages.
